2 End-to-End Sequence (User Story)

Time-ordered interaction between actors and the system, from link generation to the lender using the demo in a validation conversation.

sequenceDiagram
  autonumber
  actor Admin
  actor Borrower
  participant Sys as GCP System
  participant Prov as Sandbox Providers
  actor Lender
  Admin->>Sys: Seed demo lender + scenarios
  Sys-->>Admin: Lender-specific link (/apply/...)
  Borrower->>Sys: Open lender-branded start screen
  Borrower->>Sys: Select type/corridor, profile, consent
  Borrower->>Sys: Enter KYC / financial / supporting data + upload docs
  Borrower->>Sys: Review & Submit
  Sys->>Prov: Request normalized data (sandbox)
  Prov-->>Sys: KYC / bureau / income / supporting signals
  Sys->>Sys: Confidence scoring + risk band + reason codes
  Sys-->>Borrower: Submission confirmation
  Lender->>Sys: Login (role-based)
  Lender->>Sys: Open applicant detail
  Sys-->>Lender: Risk band, confidence, factors, flags, next step
  Lender->>Sys: Download PDF / view audit / add notes

5 Risk Translation Logic

Explainable, rule-based (not ML). Selects a scenario, scores each source's confidence, then derives the risk band, factors, flags, gaps, reason codes and a recommended next step — all versioned.

flowchart TD
  IN["Normalized GCP data
+ borrower type / corridor"]
  SEL["Scenario selection
based on type, corridor, source availability"]
  CONF["Data confidence scoring
reliability · verification · freshness ·
consistency · coverage · fraud · relevance"]
  BAND{"Risk band"}
  B1["🟢 Low Risk"]
  B2["🟡 Moderate Risk"]
  B3["🟠 Elevated Risk"]
  B4["⚪ Insufficient Data"]
  OUT["Risk output object"]
  D1["Overall confidence score/level"]
  D2["Positive factors"]
  D3["Risk flags"]
  D4["Missing-data notes"]
  D5["Recommended next step"]
  D6["Reason codes + rule version"]
  IN-->SEL-->CONF-->BAND
  BAND-->B1 & B2 & B3 & B4
  B1 & B2 & B3 & B4 --> OUT
  OUT-->D1 & D2 & D3 & D4 & D5 & D6
  NOTE["⚠️ Decision-support only —
does not approve or deny credit"]
  OUT-.->NOTE
  classDef eng fill:#1c2742,stroke:#58a6ff,color:#fff;
  classDef warn fill:#3a2d12,stroke:#d29922,color:#fff;
  class IN,SEL,CONF,OUT,D1,D2,D3,D4,D5,D6 eng; class NOTE warn;
